# Telegrab

**Telegrab** turns your Telegram account into unlimited, private cloud storage. No caps, no subscription fees, just your files on the cloud you already trust.

![License: MIT](https://img.shields.io/badge/License-MIT-emerald.svg?style=flat-square)
![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20macOS-blueviolet.svg?style=flat-square)

## ✨ Features

- **Unlimited Storage** — Powered by Telegram's cloud, no storage limits.
- **File Explorer** — Full-featured UI with drag & drop, folders, and list/grid views.
- **Media Streaming** — Watch 4K videos or listen to audio instantly without full downloads.
- **Privacy First** — Files stay between you and Telegram. No third-party servers.
- **Developer Ready** — Optional REST API for automations and LLM integrations.

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS, Framer Motion.
- **Backend**: Python 3.11+, Telethon (MTProto), FastAPI.
- **Desktop**: pywebview (WebView2 / WebKit).

## 🚀 Quick Start

1. **Download**: Grab the latest version from the [Releases](https://github.com/jithin-jz/telegrab/releases) page.
2. **API Keys**: You'll need your own Telegram API keys from [my.telegram.org](https://my.telegram.org) (API development tools).
3. **Login**: Launch the app and sign in via QR Code or Phone number.
