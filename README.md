## ✨ Features

| | Feature | Description |
|---|---|---|
| ☁️ | **Unlimited Storage** | Powered by Telegram's cloud — no storage limits, ever. |
| 📁 | **File Explorer** | Full-featured UI with drag & drop, folders, and list/grid views. |
| 🎬 | **Media Streaming** | Watch 4K videos or listen to audio instantly without full downloads. |
| 🔒 | **Privacy First** | Files stay between you and Telegram. No third-party servers. |
| 🛠️ | **Developer Ready** | Optional REST API for automations and LLM integrations. |
